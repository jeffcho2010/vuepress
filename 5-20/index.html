# IntersectionObserver

原文

優點

  • 可以很簡單的寫一些滾動動畫
  • 不管橫式直式都能共用程式

缺點

  • 會被 css 影響他真正被看到的時候
  • 動畫如果會動到元素本身，他被看到的瞬間然後往下移動，就會變成看不到，就會卡在那邊
