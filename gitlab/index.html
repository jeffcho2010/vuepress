  1. 在 gitlab 上開 project
    • 名稱：project
    所有者選 blockstudio
  2. 從 gitlab clone wp_template基礎樣板
    cd /Applications/MAMP/htdocs/display
    git clone --recursive git@gitlab.com:blockstudio/wp_template.git 2020project
    cd 2020project
  3. 用fork或其他工具選擇develop分支，確保最新狀態
  4. 執行remote.sh，提示輸入的時候輸入專案名稱project
    sh remote.sh
    這個指令一次完成下列動作：
    刪除原本的remote origin
    增加對應的remote origin
  5. Push 上 gitlab
  6. 創建資料庫名稱：blockstu_2020project
  7. 編碼選擇：utf8mb4_unicode_ci
  8. 匯入SQL：非多國語言的blockstu_basic.sql或多國語言的blockstu_basic-lang.sql
  9. 修改資料庫：選擇資料表wp_options修改siteurl與home欄位
    siteurl：http://localhost/display/2020project/main
    home：http://localhost/display/2020project
    前者是資料夾路徑
    後者是網站路徑，視情形亦可下者同上
  10. 更改wp-config.php的資料庫名稱為blockstu_2020project
  11. 更改 webpack.config.js 的 proxy為http://localhost/display/2020project
  12. 使用以下網址進入wp-admin
    http://localhost/display/2020project/main/wp-admin
  13. 登入後點擊設定->固定網址，點擊儲存變更按鈕即可使用正常的http://localhost/display/2020project/wp-admin網址登入後台
